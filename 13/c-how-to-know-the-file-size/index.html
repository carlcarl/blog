C how to know the file size

日 27 3月 2011
By carlcarl

In C & C++.

兩種方法~

1.利用 fseek(或 lseek)

FILE* f = fopen(fileName, "r+");
fseek(f, 0, SEEK_END);
size = ftell(f);
fseek(f, 0, SEEK_SET);

2.用 stat 函式

struct stat st;
stat(fileName, &st);
size = st.st_size;
