Java paintComponent paint duplicate

二 11 6月 2013
By carlcarl

In java.

tags: java

之前寫 Java 的時候遇到的小問題，在實作 JPanel 的 paintComponent 時，發現怎麼畫完後會出現多餘的重複物件。

後來找了一下資料，在 stackoverflow 發現有人有一樣的問題。解決方法也很簡單，只要在 paintComponent 一開始加上 super.paintComponent(g);，可惜的是不懂為啥不呼叫的話會出現這樣的結果就是。
