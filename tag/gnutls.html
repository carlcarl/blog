git gnuTLS error

三 22 2月 2012
By carlcarl

In 版本管理.

tags: gitgnutls

之前有台機器莫名沒辦法做「git clone https連結」，都會出現 error: GnuTLS recv error (-9): A TLS packet with unexpected length was received，查了老半天，還跟其他台的 git 比較設定，還是找不到答案，後來看到有人自己抓下來編譯好像就OK，所以我也想說自己抓 source code 下來試試看好了 XD。

去官網抓了 source code 下來，編譯並裝到 /usr/local 底下，測試了一下，果然就可以了！反正更新啥的我也沒差，所以目前就先這樣囉～。
