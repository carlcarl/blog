    javascript style.width offsetWidth

    查了一些資料，看起來 style.width 是如果 css 沒指定 element 的 width 的話，就會拿到空值，即使它本身的確佔有一個大小。這時候如果想拿到實際大小的話可以用 offsetWidth，另外要注意的是 offsetWidth 是 readonly，所以如果要設寬度，還是得從 style.width 著手。還有 這篇 是 style.width 和 offsetWidth 的效能比較，看來 offsetWidth 效能好很多，看來以後抓寬度就直接抓這個值就好了@@。
