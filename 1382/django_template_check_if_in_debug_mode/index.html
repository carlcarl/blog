    Django template check if in debug mode

    找了一下在 template 中確認是否在 debug mode 底下的方法，找了很多，不過好像都有點繞圈子，後來發現其實 Django 內建還是有簡單的方法。

    在 settings.py 裡，先確認 debug 是 True，然後再加上：

    INTERNAL_IPS = (
    '127.0.0.1',
    'localhost',
)

    之後在 template 裡就可以這樣用:

    {% if debug %}
<span>This is debug</span>
{% else %}
<span>This is production</span>
{% endif %}

    如果到 production 的話，就將 debug 設為 False。
