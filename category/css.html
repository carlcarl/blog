    Link 如果在下面的情況是 OK 的：

    <a href="#">
    <div>test</div>
</a>

    但是如果加了 float 後就會出問題：

    <a href="#">
    <div style="float:left;">test</div>
</a>

  3. 五 22 2月 2013

    bootstrap input align

    使用 bootstrap 的 input element 的時候遇到 text 和 button 位置不齊的情況，去 stackoverflow 找了一下就看到解決的辦法了：

    <div class="form-horizontal">
    <input name="search" id="search"/>
    <button class="btn">button</button>
</div>

    簡單說就是在外面那層加上 form-horizontal 這個 class 屬性，另外 input-append 雖然也可以達到類似的效果，不過由於 input element 會貼在一起，所以不建議用。
