    首先要先在 Configurator 的檔案加入下面內容 (官方 doc 裡面沒提到要從哪邊 import JSONP)：

    from pyramid.renderers import JSONP
# 這裡使用 'callback' 當作預設的 callback function 名稱
config.add_renderer('jsonp', JSONP(param_name='callback'))

    然後在 view 裡面將 renderer 改用 jsonp，這邊你還是可以加上 route_name 等參數：

  10. Python Django use MEDIA_URL in templates

    六 19 3月 2011
    By carlcarl

    In python.

    在 settings 裏面已經設好 MEDIA_URL了，可是一直不知道怎麼在 templates 中使用
    像是:

    href="{{ MEDIA_URL }}styles/base.css"

    後來看到一篇有說在 retrun render_to_response 這邊需要修改，改成這樣, 把第三個參數加上 context_instance 這句:

    return render_to_response('base/index.html', {}, context_instance =
    RequestContext(request))

    這樣就可以使用了~~。
